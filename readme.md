## Endpoints
 
### 1. `GET /usuarios`
Retorna uma lista de 5 usuários.
 
### 2. `POST /usuarios`
Cria um novo usuário. Campos obrigatórios:
- `login`
- `senha`
 
### 3. `PUT /usuarios/{id}`
Atualiza os dados de um usuário pelo ID.
 
### 4. `DELETE /usuarios/{id}`
Exclui um usuário pelo ID.
